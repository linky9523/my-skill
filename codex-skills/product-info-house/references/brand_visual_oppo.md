# OPPO Brand Visual Guidance For Product Info House

Use this reference when generating OPPO-facing product information house images, or when the user does not provide a color/style direction.

## Default Theme

- Default to a green and white visual system.
- Primary color: OPPO green.
- Background: clean white or very light warm white.
- Supporting colors: deep green, soft green tint, neutral dark gray, light gray lines.
- Use red only for risk/todo labels, and keep it restrained.

## Visual Tone

- Clean, premium, calm, and product-strategy oriented.
- Avoid cartoon, overly decorative, neon, cyberpunk, heavy gradients, or colorful magazine style.
- The image should feel suitable for internal product planning, launch alignment, white papers, and strategy decks.
- Use clear hierarchy and enough whitespace.

## Product Information House Layout

- Horizontal information house diagram.
- Roof: product positioning.
- Core row: core user value.
- Three pillars: user value, product capability, communication expression.
- Foundation: evidence and risk/todo.
- Keep the house shape recognizable; do not turn it into a generic card grid.

## Typography And Text

- Chinese text must be legible and not crowded.
- Use concise headings and short bullet points.
- Do not add new product claims, numbers, certifications, or awards beyond the provided content.
- Avoid extreme claims such as "最强", "第一", "唯一", "行业领先" unless the source material explicitly supports them.

## Image2 Prompt Requirements

When calling Image2, include:

```text
Create a horizontal product information house diagram in OPPO-inspired green and white visual style.
Use a clean white background, OPPO green roof and key lines, dark gray body text, restrained red only for risk/todo.
The layout must include: roof positioning, core value row, three pillars, foundation evidence, foundation risk/todo.
Keep all Chinese text legible and faithful to the provided content.
Do not invent product claims, parameters, certifications, awards, or extreme statements.
Make it look like a premium product strategy deliverable, not a cartoon illustration.
```

If the user provides a different brand or color direction, use the user's direction instead of this default.
