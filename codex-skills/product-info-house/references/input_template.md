# Product Info House Input Template

Use this template when the user's source material is too loose or they ask how teammates should provide information.

```text

产品版本/地区：
目标用户：
核心场景：
已确认功能/参数：
用户痛点或机会：
希望强化的心智：
已有证据：
待确认信息：
不能说/需避开的表达：
```
