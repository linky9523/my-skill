# One-Image Product Infographic Patterns

## Poster-Like Sheet Order

1. KV: product name, positioning, slogan, co-brand, product hero visual.
2. Core promise: one strong sentence that translates specs into user value.
3. Highlights/KSP: 3 strongest cards, each with title, benefit, proof, and image placeholder.
4. Main capability sections: usually 2-3 compact sections after Highlights. Combine secondary specs instead of expanding every website section.
5. Optional product-experience ending: only include if it is a product-planning message, not sales or channel information.

Do not include price, package bundle, preorder, QR, sale time, channel, or purchase mechanics by default. Keep the first sheet close to a normal 一图读懂 length, not a website-long page.

## Copy Hierarchy

- Level 1 section title: tells the reader why the section matters.
- Level 2 selling-point title: names the capability and gives a memorable benefit.
- Level 3 proof line: parameters, mechanisms, feature names, or scenario proof.
- Body copy: only use when the mechanism or brand story matters.
- Notes: non-copy instructions for designer or planner.

## Rough Layout Patterns

- Hero/KV: one large merged band; visual direction should mention product angle, logo, and background need.
- Core KSP: one large lead card plus two to four supporting cards.
- Sample-heavy section: use a wide image strip or 2-column sample cards.
- Capability detail: use left text + right product/CG/material cue, or alternating cards.
- All-round experience: use a 2x2 or 2x3 card grid.
- Length control: visible poster canvas should be KV + 3-4 major sections. Each section should show at most 3 visible cards; move the rest to the copy/material list.

## Planner Writing Rules

- Start from user benefit, then attach parameters.
- Keep one selling point per card.
- Use official names exactly, but rewrite generic spec lists into readable benefits.
- Mark `待确认` if a claim lacks source support.
- Include material needs such as `产品 KV`, `爆炸图`, `样张拼图`, `器件 CG`, `系统界面`, `屏幕特写`, `芯片/电池 CG`.
