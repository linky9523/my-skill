# Official Website Remark Patterns

## Core Style

- Use sober compliance wording, not persuasive copy.
- Start from the exact claim: parameter, function, sample, material, accessory, test result, or availability condition.
- Keep one row's remark tied to that row only. Do not add broad all-page disclaimers into every row.
- Use placeholders such as `xxx`, `xx`, or product names only when the source row does not provide confirmed values.
- If a row already has a valid remark, keep it and only add missing constraints when needed.
- Use `/` when the row is only a planning label, section divider, or internal material note with no public-facing claim.

## Common Categories

### General Product And Design

Use when the row discusses product appearance, structure, color, material, size, weight, or process:

`产品图片仅供参考，请以实物为准。产品规格、外观细节、材质及工艺可能因生产批次、供应商、测量方法等因素略有差异，请以产品实际为准。`

### Display And Eye Care

Use when the row discusses brightness, refresh rate, dimming, eye care, screen size, bezels, or touch:

`屏幕相关数据来源于 OPPO 实验室，实际显示效果可能因使用环境、设备状态、软件版本、测试方法及个人使用习惯不同而存在差异，请以实际体验为准。护眼相关功能非医疗器械功能，不具有治疗作用。`

### Camera Hardware And Photo Effects

Use when the row discusses lenses, sensors, pixels, focal lengths, aperture, OIS, zoom, color, HDR, or imaging algorithms:

`影像规格、焦段、进光量、动态范围等数据来源于 OPPO 实验室或供应商测试。实际拍摄效果可能因拍摄环境、拍摄对象、设备状态、软件版本及拍摄模式不同而存在差异，请以实际体验为准。`

### Samples And Materials

Use when the row discusses sample photos, videos, UI screenshots, material demos, CG, generated images, or layout effects:

`页面样张、视频、产品界面、结构示意图及素材效果仅供参考，可能经过裁切、压缩、合成或创意化呈现，请以实际拍摄效果、产品界面和最终上线页面为准。`

### Zoom

Use when the row discusses optical zoom, optical-quality zoom, digital zoom, telephoto, equivalent focal length, teleconverter, or long-distance shooting:

`变焦、等效焦段及远摄效果可能因拍摄距离、光线、模式、算法处理及设备状态不同而存在差异。光学品质变焦、数字变焦等能力请以产品实际支持情况为准。`

### Video, Log, HDR, And Live Photo

Use when the row discusses 4K, 8K, HDR, Log, LUT, live photo, motion photo, stabilization, or video color:

`视频、HDR、Log、LUT、实况照片等功能的可用规格和效果与拍摄模式、软件版本、存储空间、环境光及设备状态有关，请以实际体验为准。`

### AI And Software

Use when the row discusses AI, ColorOS, UI, albums, assistant features, recognition, editing, or system smoothness:

`以上软件界面和功能仅作展示。AI 及系统相关能力可能因软件版本、地区、账号登录状态、网络环境、适配应用及个人使用习惯不同而存在差异，请以实际版本功能及界面为准。`

### Battery And Charging

Use when the row discusses capacity, battery life, cycle life, wired charging, wireless charging, reverse charging, or low-temperature charging:

`电池容量、续航和充电数据来源于 OPPO 实验室或供应商测试，实际表现可能因网络环境、使用方式、充电条件、设备温度、电池老化及其他因素不同而存在差异，请以实际体验为准。`

### Waterproofing And Durability

Use when the row discusses IP ratings, waterproofing, dust resistance, drop resistance, scratch resistance, or harsh tests:

`防水、防尘、抗摔、抗划等数据来源于受控实验室条件。相关防护能力并非永久有效，可能因日常磨损而下降；请勿在潮湿状态下充电，因进液、跌落、碰撞等导致的损坏请以保修政策为准。`

### Performance And Storage

Use when the row discusses chipsets, performance uplift, benchmark scores, RAM, storage, smoothness, thermal data, or gaming:

`性能、跑分、存储、散热及游戏表现数据来源于 OPPO 实验室或官方测试，实际表现可能因测试环境、软件版本、内存规格、网络环境、应用适配及个人使用习惯不同而存在差异。`

### Connectivity, Sound, And Third Parties

Use when the row discusses cellular network, satellite/offline communication, NFC, cross-device features, volume, audio, third-party apps, or third-party content:

`通信、音频、协同及第三方服务的功能表现可能因地区、运营商、网络环境、设备状态、软件版本、应用适配及第三方服务政策不同而存在差异，请以实际支持情况为准。`
