# 小布助手

小布助手是团队 Skill 包的入口导航助手。它会主动介绍团队里已有的 Skill 能做什么，并根据用户的任务推荐下一步该用哪个 Skill。

它适合给不熟悉 AI 或 Skill 的业务同事使用。用户不需要懂 Prompt，也不需要一开始就知道该选哪个 Skill，只要说出自己想做什么、手上有什么资料、希望输出成什么形式，小布助手就会帮忙拆解。

## 小布助手可以帮你做什么

- 介绍团队已有 Skill 包分别能做什么。
- 根据业务任务推荐合适的 Skill。
- 告诉你需要准备哪些输入材料。
- 把模糊需求改写成可直接复制使用的 Prompt。
- 提醒你如何检查输出结果是否可信、可用、可交付。

## 适合什么时候用

- 第一次使用团队 Skill，不知道从哪里开始。
- 想知道“这些 Skill 到底能帮我做什么”。
- 有业务任务，但不知道该调用哪个 Skill。
- 想把复杂任务拆成几个步骤完成。
- 想确认自己的资料是否足够。
- 想知道输出后该怎么继续追问和修改。

## 不适合什么时候用

- 已经明确知道要用哪个业务 Skill，并且要直接产出交付件。
- 要创建或修改一个新 Skill。
- 要建立项目知识库。
- 只是问一个简单问题，不需要选择 Skill。

## 可以这样开始

```text
你好小布助手，我想做一个新品官网框架，但不知道该用哪个 Skill。请先介绍一下相关 Skill 能做什么，再帮我推荐下一步。
```

也可以这样问：

```text
小布助手，我有一份卖点信息表，想转成 PPT 内容。请帮我判断该用哪个 Skill，并把我的需求改写成可以直接执行的 Prompt。
```

## 常见输出

小布助手通常会输出：

1. 小布助手自我介绍
2. 推荐使用的 Skill
3. 推荐理由
4. 需要准备的材料
5. 可直接复制使用的 Prompt
6. 输出后的检查方式
7. 下一步建议
