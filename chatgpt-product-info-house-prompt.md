# ChatGPT 版：一句话生成产品信息屋

给只用 ChatGPT 的同事，复制下面这一段即可。后面直接粘产品资料、会议纪要或表格。

```text
请把下面的产品资料整理成“产品信息屋”，用于产品策划内部对齐和汇报图生成。

要求：
1. 只基于我提供的信息，不要编造参数、测试数据、认证或竞品结论。
2. 输出要简短，适合放进一张图里。
3. 自动整理为：产品名称、产品定位、核心价值、三根信息支柱、证据支撑、风险与待补。
4. 如果缺少证据，请写进“风险与待补”，不要包装成确定结论。
5. 最后只输出 JSON，方便我复制进产品信息屋生成器。

JSON 格式如下：
{
  "productName": "产品名称",
  "positioning": "一句话产品定位",
  "coreValue": "用户为什么选择它",
  "pillars": [
    {
      "title": "用户价值",
      "points": ["短要点 1", "短要点 2", "短要点 3"]
    },
    {
      "title": "产品能力",
      "points": ["短要点 1", "短要点 2", "短要点 3"]
    },
    {
      "title": "传播表达",
      "points": ["短要点 1", "短要点 2", "短要点 3"]
    }
  ],
  "evidence": ["证据 1", "证据 2", "证据 3"],
  "risks": ["风险或待补 1", "风险或待补 2", "风险或待补 3"]
}

产品资料如下：
```

## 同事使用方式

1. 复制上面整段提示词。
2. 把产品资料粘在“产品资料如下：”后面。
3. 让 ChatGPT 输出 JSON。
4. 把 JSON 粘进 `index.html` 的“产品信息 / 表格”输入框。
5. 点击“生成信息屋”，再微调并导出 PNG。

## 更短口令

如果同事已经知道流程，也可以只用这一句：

```text
请把下面资料整理成产品信息屋 JSON：产品名称、产品定位、核心价值、三根信息支柱、证据支撑、风险与待补。只基于资料，不编造，缺证据写待补。资料如下：
```
